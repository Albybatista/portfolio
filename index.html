I'm Alby


As a creative, solutions-oriented software developer, my favorite place is a blank text editor. Being a kinesthetic learner, all I’ve known from a young age has been how to build and break things in order to just recreate them. One of the most gratifying feelings is coming up with a creative solution to a problem while collaborating with others. Currently, what I enjoy most as a developer is the ability to have a hands on approach to how things work and how they can be improved.

My experiences with people, in sales, business, and web development allows me to be a multifaceted devloper where I can maximize overall company efficiency. Being a great communicator and having a people-first mentality has paved the way for most of my successes. This outlook has allowed me to think like a user, the experience, and finally, as a business as well.

Builder's Block

Destination Ratr

A collaborative project and single-page application where users can rate and post their travel experiences in a blog-like manner. This was built with create-react-app on the front-end, PHP on the back-end. We created our own API and tables to make this possible with PostgreSQL as the database. The PHP back-end was deployed with Heroku and the front-end was deployed with Netlify.

Frontend - React   Backend - PHP   Database - PostgreSQL

Scriven

Notes app that integrates code blocks with regular text. This app was built with the PERN stack which includes Express and Node.js using React, and PostgreSQL as the database.

Frontend - React   Backend - Express/Node.js   Database - PostgreSQL

Iconic Baseball

A multi-page application with multiple routes where users can post images that captured baseball’s most iconic moments. The landing page features a carousel of of three images in order to ease the eyes at first glance. The app features a fully self-built database with images, dates, and descriptions that describe the events that occured. This app was built with Express and Node.js using JavaScript with EJS templating, and MongoDB as the database.

Frontend - JavaScript/EJS   Backend - Express/Node.js   Database - MongoDB
